## Usage Guidelines

### Font Family

- **Sans Serif (Inter)**: Use for all UI elements, body text, and headings
- **Monospace**: Use only for code snippets, file paths, and technical content

### Font Size

- **text-xs/sm**: Use for supplementary information, captions, error messages
- **text-base**: Default for most body content and navigation
- **text-lg**: Use for emphasized body text and post metadata
- **text-xl/2xl**: Use for page headings and section titles
- **text-3xl/4xl**: Use sparingly for hero headings and marketing content

### Font Weight

- **font-normal**: Default for body text and most content
- **font-medium**: Use for navigation items and subtle emphasis
- **font-semibold**: Use for buttons, form labels, and important UI elements
- **font-bold**: Use for headings and strong emphasis

### Line Height

- **leading-tight**: Use for headings and compact layouts
- **leading-normal**: Default for most content
- **leading-relaxed**: Use for long-form reading content

## Accessibility

- All text maintains WCAG 2.1 AA contrast ratios
- Font sizes are responsive and scale appropriately
- Line heights ensure comfortable reading across all devices
- Font weights provide clear hierarchy without relying solely on color
