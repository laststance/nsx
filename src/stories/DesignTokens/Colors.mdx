## Usage Guidelines

- **Primary (Green)**: Use for primary actions, success states, and brand elements
- **Secondary (Blue)**: Use for secondary actions and informational elements
- **Danger (Red)**: Use for error states, destructive actions, and critical warnings
- **Gray/Neutral**: Use for text, borders, backgrounds, and general UI elements
- **Semantic utilities**: Use custom utilities (bg-primary, text-color-primary, etc.) for automatic theme switching

## Color Accessibility

All color combinations follow WCAG 2.1 AA accessibility guidelines for contrast ratios:

- Large text (18px+): minimum 3:1 contrast ratio
- Normal text: minimum 4.5:1 contrast ratio
- Interactive elements: minimum 3:1 contrast ratio
